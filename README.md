# RadioTransfer
A Minecraft mod which implements the development of radios to transfer resources, power, and data.

## How to intitialize environment
(instructions from the [Forge Documentation](https://mcforge.readthedocs.io/en/latest/gettingstarted))
- Clone this repo, and cd to the project directory
- run `chmod +x gradlew`
- make sure you have gradle installed, and run `./gradlew setupDecompWorkspace`
- run `gradlew genIntellijRuns`, open the folder as a project, and import the build.gradle file
