# com.rlapcs.radiotransfer.RadioTransfer
A Minecraft mod which implements the development of radios to transfer resources, power, and data.

## How to intitialize environment
- Clone this repo, and cd to the project directory
- Follow steps 4-8 in the [Forge Documentation](https://mcforge.readthedocs.io/en/latest/gettingstarted/#terminal-free-intellij-idea-configuration) (Section: "Terminal-free IntelliJ IDEA configuration)
